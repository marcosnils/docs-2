## What's a UiHook

A **UiHook** is the name we use for the HTTP endpoint of an Integration. We use the name **UiHook** because it's similar to a Webhook, but based on user interfaces.

- It's another name for a Wehbook
- It's the name given for the HTTP endpoint of the ZEIT Integration
- It's another name for a ZEIT Integration
- It's the name given for the installation of a ZEIT Integration
