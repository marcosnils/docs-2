## MongoDB Atlas

This Integration allows you to connect with a MongoDB Atlas account. After that, it can link a database cluster to any project in a ZEIT account.

When linked, the Integration will set the `MONGO_URL` environment variable related to the selected database cluster.

Then, the user can deploy without having to configure database users, setup ip-filtering or create ZEIT secrets manually.

Taking a look at the [code on GitHub](https://github.com/zeit/integrations-gallery/tree/master/mongodb-atlas) will help you understand how to build Integrations that integrate with MongoDB Atlas using the UiHook.
