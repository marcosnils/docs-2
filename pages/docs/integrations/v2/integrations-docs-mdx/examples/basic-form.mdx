## Basic Form

This example shows how to receive user inputs and manage different actions. In this Integration, the user can set two values and reset them using a form.

Taking a look at the [code on GitHub](https://github.com/zeit/integrations-gallery/tree/master/basic-form) will help you understand how to build form Integrations using the UiHook.
