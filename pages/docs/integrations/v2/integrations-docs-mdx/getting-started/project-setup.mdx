## Project Setup

### Install Node.js and Now CLI

<Note>
  If you already have the latest stable versions of Node.js and{' '}
  <GenericLink href="/now">Now CLI</GenericLink>. You can skip this step.
</Note>

- Download and install [Node.js](https://nodejs.org/en/)
- Download and install [Now Desktop](/download) (which comes with Now CLI)

If you prefer, you can also install the [Now CLI](/now) using your package manager of your choice.

Using [Yarn](https://yarnpkg.com/):

<TerminalInput>yarn add global now</TerminalInput>
<Caption>Installing Now globally with Yarn.</Caption>

Using [npm](https://www.npmjs.com/):

<TerminalInput>npm install -g now</TerminalInput>
<Caption>Installing Now globally with npm.</Caption>

### Add a package.json File

To use modules inside of your UiHook, you need to create a `package.json` file in the **root directory** of your UiHook. Create one with the following content:

```json
{
  "name": "my-ui-hook",
  "dependencies": {
    "@zeit/integration-utils": "^1.0.0"
  }
}
```

<Caption>
  An example <InlineCode>package.json</InlineCode> file for a ZEIT Integration.
</Caption>

In the example `package.json` file above, the `@zeit/integration-utils` module is installed, this allows you to use the [Component Catalog](/docs/integrations#component-catalog/page) to create your UiHook.

Once you have created your `package.json` file, use the applicable command below to install the dependencies.

If installed with Yarn:

<TerminalInput>yarn</TerminalInput>
<Caption>Installing project dependencies with Yarn.</Caption>

If installed with npm:

<TerminalInput>npm install</TerminalInput>
<Caption>Installing project dependencies with npm.</Caption>

### Add now.json

Since this UiHook is deployed to Now, a `now.json` file is used to provide minimal configuration for the deployment. You should use a `now.json` file when running your Integration locally with `now dev`.

Create a `now.json` file and add the following content:

```json
{
  "name": "my-ui-hook",
  "version": 2,
  "builds": [{ "src": "index.js", "use": "@now/node" }]
}
```

<Caption>
  An example <InlineCode>now.json</InlineCode> file for a ZEIT Integration.
</Caption>

That's all that is required for the initial project setup, next you will create your first UiHook.
